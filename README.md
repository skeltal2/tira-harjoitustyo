## Lyhyt käyttöohje

Asenna riippuvuudet komennolla:
```
poetry install
```
Aloita ohjelma komennolla:
```
poetry run invoke start
```

Koko käyttöohjeen voi katsoa [tästä](https://github.com/skeltal2/tira-harjoitustyo/blob/main/Dokumentaatio/kayttoohje.md).
