## Käyttöohje
Asenna riippuvuudet:
```
poetry install
```
Aloita peli:
```
poetry run invoke start
```
Koska pelillä ei vielä ole riippuvuuksie sen voi myös aloittaa suorittamalla:
```
python3 src/index.py
```
